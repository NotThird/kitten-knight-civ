Policy: task quotas (AI plan bias)
These multipliers bias the colony-wide worker plan. 0 = never (unless safety/emergency), 1 = default, 2 = strongly prefer. Great for nudging specialization.
Role quotas (gentle specialization targets)
Optional. If you set a quota above 0, the AI will try to keep that many kittens in the role (still overridden by safety rules/emergencies).
